# ホシドタロ — 홈 재구성 + 사운드 강화 + 서비스 고도화 (v9)

타입 체크(strict) + 사운드 케이스 커버리지 + BGM 코드 정합성 검증 통과.

═══════════════════════════════════
## ① 홈 카테고리 재구성
═══════════════════════════════════

**문제**: 기능이 늘면서 홈에 링크가 세로로 계속 붙어 지저분했음
(컬렉션→compat→seikaku→weekly→lucky→... 무한 나열).

**해결**: 4개 카테고리 섹션 + 2열 그리드로 정리.
```
히어로(워드마크)
메인 오미쿠지 2장  ← 간판: 今日の運勢 / する・しない
스트릭 + 今日の一枚
① 占う          → 一枚引きタロット
② 相性・診断     → 相性占い / 性格診断 / 相性チェッカー / 相性ランキング
③ 毎日チェック   → 今週の運勢 / ラッキーアイテム / 月別運勢
[광고]
④ コレクション・ガイド → コレクション / 占いガイド
앱 설치 카드
```
각 섹션은 금색 제목 + 구분선, 타일은 아이콘+제목+한줄설명+화살표.
주력 기능(一枚引き·相性占い)은 `accent`로 금색 강조.

**파일**:
| 파일 | 상태 |
|---|---|
| `src/app/page.tsx` | 덮어쓰기 (카테고리 재구성) |
| `src/components/home/ServiceTile.tsx` | 신규 (재사용 타일, 탭 사운드 내장) |

※ 기존 홈의 SEO 텍스트 블록은 ④ 아래 또는 푸터 위에 유지하고 싶으면
   그대로 두셔도 됩니다(이 버전은 간결화를 위해 가이드 링크를 ServiceTile로 흡수).
   SEO 텍스트를 남기려면 기존 `<section>...占いガイド...</section>` 블록을
   ④ 섹션 아래에 붙여넣으세요.

## ② 사운드 강화
═══════════════════════════════════

**효과음 8종으로 확장** (기존 4 + 신규 4):
- 기존: tap(버튼) / shuffle(셔플) / reveal(결과) / star(레어)
- 신규: **select**(선택 확정, tap보다 단단한 2톤) / **flip**(카드 뒤집기, 나무 두드림)
  / **back**(뒤로, 하강 2톤) / **chime**(화면 진입, 잔잔한 상승 아르페지오)

**BGM 대폭 개선**:
- 단조로운 3음 고정 화음 → **4코드 진행(Am→F→C→G)을 8초마다 부드럽게 순환**
  (1.5초 글라이드로 자연스럽게 전환). 로우패스 필터로 "밤하늘" 질감.
- 여전히 기본 OFF + 우상단 토글 + localStorage 기억.

**적용 지점** (이번 패치에 이미 반영):
- 홈: 오미쿠지·타일 탭 → tap
- 一枚引き: 카드백→앞면 **뒤집기 연출** + flip/reveal
- 相性占い: 관계 선택 → select
- 今週の運勢/ラッキー: 결과 로드 완료 → reveal/chime

**파일**:
| 파일 | 상태 |
|---|---|
| `src/lib/useSound.ts` | 덮어쓰기 (8종 확장) |
| `src/components/SoundControl.tsx` | 덮어쓰기 (BGM 코드 진행) |

## ③ 서비스 고도화
═══════════════════════════════════

- **一枚引きタロット**: 뽑기 결과가 **카드 뒷면에서 앞면으로 3D 뒤집히는 연출**
  (perspective + rotateY). 뒤집힘과 동시에 flip→reveal 사운드로 "뽑았다"는 손맛 강화.
- **相性占い**: 관계 선택에 확정음(select) — 단계 진행의 리듬감.
- **今週の運勢 / ラッキーアイテム**: 결과가 뜰 때 부드러운 사운드로 완료감.
- (홈) **ServiceTile**: 모든 이동에 tap 사운드가 기본 내장 → 사이트 전체 통일감.

**파일**:
| 파일 | 상태 |
|---|---|
| `src/app/hitokoto/page.tsx` | 덮어쓰기 (뒤집기 연출) |
| `src/app/compat/page.tsx` | 덮어쓰기 (select 사운드) |
| `src/app/weekly/page.tsx` | 덮어쓰기 (reveal) |
| `src/app/lucky/page.tsx` | 덮어쓰기 (chime) |

## 설계 메모
- ServiceTile은 useSound를 내장해, 새 기능 추가 시 이 컴포넌트만 쓰면 사운드가 자동 적용.
- BGM/효과음 모두 Web Audio 생성이라 여전히 오디오 파일 0개(용량·로딩 0).
- 사운드는 전부 SoundControl 토글로 끌 수 있어 접근성/사용자 선택권 보장.
- 홈이 길어졌지만 섹션 구분으로 스캔성이 좋아짐. 필요시 ③④ 섹션을 접이식(details)으로
  만들 수도 있으니 원하시면 요청 주세요.
